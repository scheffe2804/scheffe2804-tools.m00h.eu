6. Einzelne Tools dieser Website

PrivateBin (bin.m00h.eu)

PrivateBin verschlüsselt Inhalte vollständig im Browser. Der Server empfängt und speichert ausschließlich verschlüsselte Daten und hat keinen Zugriff auf den Klartext. Es werden keine Nutzerdaten über den normalen Serverbetrieb hinaus gespeichert.

Shlink (go.m00h.eu)

Shlink speichert beim Aufruf eines Kurzlinks die anonymisierte IP-Adresse, Klickzeitpunkt und Referrer zur Klickauswertung. Diese Daten werden ausschließlich auf eigener Infrastruktur gespeichert.
